# Calculadora Financiamento

# 📄 Descrição
Uma aplicação desenvolvida em Flutter para simular financiamentos, permitindo calcular o valor das parcelas com base no valor do crédito, taxa de juros e prazo. O app retorna informações como valor da parcela, total pago e juros acumulados, facilitando o planejamento financeiro.
